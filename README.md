# igt

## Sample web-app using following technologies: 

- anugularjs v.1.5.6, 
- spring (Spring-Boot): REST, annotation configuration
- DB: JPA data, Hibernate, QueryDSL

## Utilites used:

- Swagger, 
- Dozer, 
- Lombok

## JDK

- Java 8

## Testing

- JUnit
- Spock tests 

## Build and run

- maven:

        mvn spring-boot:run
